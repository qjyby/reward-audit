## 货币汇率参考

| 换算关系 | 比例 | 备注 |
|---------|------|------|
| 筹码 → MS | 1 筹码 = 0.43~0.45 MS | 约等于 2.3 筹码 = 1 MS |
| 恒晶石 → MS | 1000 恒晶石 = 20 MS | 高价值道具替代品 |
| 超星灵药精华 → MS | 1 个 = 1.35 MS | 冰雪派对奖励中的核心道具 |
| 日常资源 → MS | 根据等级而定 | 不同的材料/装备有不同的价值 |

**转换示例**：
- 40 筹码 = 40 × 0.43 = 17.2 MS（四舍五入到 17 MS）
- 5 个超星灵药精华 = 5 × 1.35 = 6.75 MS
- 排行榜 11-20 名奖励 = 10 × 1.35 = 13.5 MS

## 定价策略参考

### 价值比标准范围

- **超星灵药精华**：1.3-1.4 MS/个（标准定价）
- **高等级道具**：0.45-0.5 MS/个（不应低于 0.3）
- **中等道具**：0.3-0.4 MS/个
- **低等道具**：0.1-0.2 MS/个
- **保底替代品**：通常是标准道具的 60-80%

### 异常定价检查

如果商品价值比偏离标准：
- ✅ 偏高（超过标准 20%）：需要说明稀有性理由
- ❌ 偏低（低于标准 20%）：需要检查是否错误
- ⚠️ 极端异常（低于标准 50%）：**必须修改或说明**

## 审核数据检查清单

每个奖励配置表都应该检查：

- [ ] **完整性**：所有货币/道具都有价值评估，无空白或 0
- [ ] **一致性**：同类型奖励的价值比保持在标准范围内
- [ ] **合理性**：奖励总量是否与活动周期和目标玩家群体匹配
- [ ] **公平性**：不同难度/等级的奖励梯度是否合理
- [ ] **成本**：汇总后的总成本是否在可控范围内
- [ ] **备注**：异常定价是否有说明理由
