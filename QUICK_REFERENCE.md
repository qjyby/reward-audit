# Reward Audit Skill - 快速参考卡片

## 一句话说明
用 AI 审核游戏奖励配置，自动找错误、生成报告。

---

## 最快上手（3步）

### 1️⃣ 准备你的 Excel 文件

```
【游戏名】活动奖励配置.xlsx
├── Sheet1 - 日常任务
├── Sheet2 - 排行榜
└── Sheet3 - 商店
```

### 2️⃣ 告诉 AI

```
这是 XXX 活动的奖励配置，请帮我审核。
（上传 Excel 文件）
```

### 3️⃣ 收到报告

```
✓ 自动分析所有工作表
✓ 找出 7 大常见错误
✓ 生成带截图的 Word 报告
✓ 给出具体修改建议
```

---

## 检查的 7 大错误类型

| 错误类型 | 例子 | 影响 |
|---------|------|------|
| **计算错误** | 总数 ≠ 各项加总 | 成本核算不准 |
| **逻辑错误** | 条件与奖励不匹配 | 玩家困惑 |
| **缺失** | 某个档位没奖励 | 玩家体验差 |
| **重复** | 同一目标被奖励多次 | 经济失控 |
| **无上限** | 可重复领取无上限 | 通货膨胀 |
| **倒挂** | 低难度反而高奖励 | 不公平 |
| **汇率错** | 不同货币换算错误 | 价值错估 |

│   └── 修改建议：应该怎么改
│
└── 📋 快速检查清单
```

## 真实案例：【魔域】冰雪派对

### 发现问题

| 问题 | 严重度 | 后果 |
|------|--------|------|
| 价值列全为 0 | 建议 | 成本算不清 |
| 配置不一致 | 建议 | 理解混乱 |

| 排行榜断层 | 建议 | 参与度低 |

```
修复前：总成本有缺失，无法评估

## 常见问题（FAQ）

### Q1: 需要我手动分析吗？
**A:** 不需要。AI 全自动分析，你只需上传文件。

### Q2: 报告能改吗？
**A:** 能。生成的是标准 Word 文档，可以自由编辑。

### Q3: 支持其他游戏吗？
**A:** 支持。只需调整 `references/game_economy.md` 里的汇率和道具价值即可。

### Q4: 卡住了怎么办？
**A:** 等 1-2 分钟再试。或分两步：先生成截图，再生成报告。

### Q5: 报告里没有图片？
**A:** 确保用 `gen_audit_report_v2.py`，不是旧版本。

## 使用场景

✅ **适合**

- 补丁更新前的数值复审

❌ **不适合**
